# Banking Automation Project

This project demonstrates:
- Selenium automation using Java
- TestNG framework
- Page Object Model (POM)
- Extent Report Integration
- Screenshot Capturing for Pass/Fail
- Maven Project Setup

## Test Scenario:
Automated Banking Login Test:
- Launch browser
- Navigate to login page
- Enter Username/Password
- Validate Dashboard title
- Generate Extent Report with screenshots

## Tools Used
- Selenium WebDriver
- Java 17/20
- TestNG
- Maven
- Extent Reports
- GitHub Version Control
